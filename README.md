# SandwichShopProject
This project is a Java console based ordering system for a fictional business called Norcoâ€™s Famous Sandwich Shop. The program allows users to view a sandwich menu, place orders by selecting items and quantities, schedule a delivery appointment within business hours, and complete checkout. During checkout, the program calculates tax, displays an order summary, and saves the order details to a text file titled order_summary.

FEATURES: 
1. Displays a menu of sandwiches and prices
2. Allows users to select sandwiches and quantities
3. Calculates subtotal, tax, and total cost
4. Allows delivery appointment scheduling
5. Saves order summary to a text file

HOW TO RUN:
1. Open terminal
2. Compile project through "javac SandwichShopProject.java"
3. Run program using "java SandwichShopProject"
